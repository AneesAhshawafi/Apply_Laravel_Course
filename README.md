About The Project
This repository contains the full source code and practical applications developed during the Laravel 12: From Zero to Mastery course. The project serves as a comprehensive journey through the Laravel ecosystem, focusing on building scalable web applications using industry-standard practices.
Course Link: [Laravel 12 Full Course Playlist](https://youtube.com/playlist?list=PL6XRLlEsQ_7Xy0fwWHhmo5H_RCI1RbtqG&si=wBotL1ana2yCa3yT)

How to Install and Run
1. Clone the repository:
   `git clone [REPO_URL]`
2. Install dependencies:
   `composer install`
3. Create your environment file:
   `cp .env.example .env`
4. Generate app key:
   `php artisan key:generate`
5. Set up your database in `.env` then migrate:
   `php artisan migrate`
6. Start the server:
   `php artisan serve`
